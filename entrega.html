e. Obtener una lista de precios de los productos y sus categorías (SIN JOIN - ANSI SQL 89).

SELECT ListPrice AS Precio, ( SELECT Name FROM SalesLT.ProductCategory WHERE SalesLT.ProductCategory.ProductCategoryID = SalesLT.Product.ProductCategoryID ) AS 'Categoría' FROM SalesLT.Product

g. El depto. de ventas desea armar ofertas para todos los productos que correspondan con los modelos Mountain-100 y Classic Vest

SELECT SalesLT.Product.Name, SalesLT.ProductModel.ProductModelID, SalesLT.ProductModel.Name AS Model, SalesLT.Product.ListPrice FROM SalesLT.Product , SalesLT.ProductModel WHERE SalesLT.ProductModel.Name IN('Mountain-100', 'Classic Vest')

h. Obtener la lista de todos los clientes ordenados por apellido

SELECT * FROM SalesLT.Customer ORDER BY LastName ASC

i. Obtener la lista de todos los clientes ordenados por apellido y luego por nombre usando alias en tablas y campos

SELECT FirstName AS Nombre, LastName AS Apellido FROM SalesLT.Customer ORDER BY LastName ASC, FirstName ASC

j. Los primeros 5 clientes ordenados por apellido y luego por nombre

SELECT TOP 5 FirstName AS Nombre, LastName AS Apellido FROM SalesLT.Customer ORDER BY LastName ASC, FirstName ASC
